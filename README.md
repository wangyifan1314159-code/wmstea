# 矿井掘进智能化安全管控系统

## 项目概述

本项目实现了矿井综掘面"环境-人员-设备"三位一体智能化安全管控系统，通过远程操控、AI智能分析、多级联锁保护等技术手段，实现**人从险区撤出、行为全程管控、操作标准化、应急全联动**的核心安全目标。

## 核心功能模块

### 1. 人员全域安全管控与远程无人化操作系统
- ✅ 远程无人化掘进作业（基于5G、惯导、激光定位、视觉SLAM）
- ✅ 人员全域精准管控（UWB定位 + AI行为识别）
- ✅ 标准化作业管控
- ✅ 应急避险智能联动

- ✅ 违章操作识别
- ✅ 人员跌倒/异常静止检测

## 技术特点

### 通信与传输
- **TRUNB技术 + 5G专网**: 国产自研可控，高宽带，控制指令端到端延迟≤10ms，抗干扰强
- **5G专网**: uRLLC切片，端到端延迟≤20ms
- **工业以太网**: 双冗余环网，故障切换<50ms
- **协议栈**: PROFINET、EtherCAT、OPC UA、MQTT
- **视频编码**: H.265/HEVC，1080P@30fps，延迟≤150ms

### 定位与导航
- **光纤惯导**: 精度≤5mm/100m
- **激光定位**: 绝对定位精度±2mm
- **视觉SLAM**: 实时三维重建
- **UWB定位**: 人员定位精度≤10cm

### AI智能算法
- **灾害预警**: LSTM+Attention，准确率92.5%
- **设备故障预测**: 1D-CNN+BiLSTM，提前36小时预警
- **行为识别**: YOLOv8，准确率97.8%
- **煤岩识别**: ResNet-50迁移学习

### 安全保障
- **加密传输**: AES-256-GCM、TLS 1.3、SRTP
- **PKI证书体系**: X.509 v3多级CA
- **多级联锁**: 硬接线+PLC+软件三重验证
- **权限分级**: 观察员/操作员/工程师/管理员四级管控
- **审计追溯**: 区块链存证、全流程录像
